# Rewrite Everything Summary

This package rewrites the outward-facing documentation, proposals, and website copy so they all tell the same company story.

## Core shift
Old framing:
- portable trust
- machine counterparties
- install preflight as the main wedge
- broad software trust / score / graph language

New framing:
- trust before high-risk action
- protocol-grade trust substrate
- control layer between authentication and execution
- actor + authority + policy + exact action context
- replay resistance + one-time consumption + immutable events
- accountable signoff when policy requires named human ownership
- Emilia Eye as a lightweight warning layer, not a decision layer
