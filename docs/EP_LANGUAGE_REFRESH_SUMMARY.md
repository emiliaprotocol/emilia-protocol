# EP Language Refresh Summary

This repo was updated to align the language with what EP now actually accomplishes in code.

## Core shift
Old framing:
- portable trust for agents, software, and machine counterparties
- broad trust infrastructure
- MCP-first company identity

New framing:
- trust before high-risk action
- protocol-grade trust substrate
- control layer between authentication and execution
- actor + authority + policy + exact action context
- replay resistance + one-time consumption + immutable events
