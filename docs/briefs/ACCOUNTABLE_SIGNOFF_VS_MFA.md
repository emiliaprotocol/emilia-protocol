# Accountable Signoff vs MFA

## MFA
MFA proves user presence.

## Accountable Signoff
Accountable Signoff proves action-specific responsibility.

It binds:
- named responsible human
- exact action
- target
- policy
- assurance method
- time window
- one-time execution semantics

## Why that matters
For government, financial, enterprise, and agent workflows, the hard problem is not only whether a user is present. It is whether a named human is willing and authorized to assume responsibility for the exact high-risk action before it executes.
