# OpenAI Cybersecurity Grant — Submission Instructions

**Form URL**: https://openai.com/form/cybersecurity-grant-program/
(verify current URL — OpenAI sometimes moves form locations; if 404,
search "OpenAI cybersecurity grant" and follow the latest blog post link)

**Cost**: Free
**Award**: $10K – $1M as cash and/or OpenAI API credits
**Cycle**: Quarterly review

## Steps

1. Go to https://openai.com/form/cybersecurity-grant-program/
2. Sign in with `iman@emiliaprotocol.ai` if prompted
3. Fill the web form fields, pasting from `application.md`:
   - **Project name** → section 1
   - **One-line description** → section 2
   - **Problem statement** → section 3
   - **Approach / solution** → section 4
   - **Why a fit for OpenAI's program** → section 5
   - **Use of funds / budget** → section 6
   - **Team** → section 7
   - **Links / public artifacts** → section 8
4. Submit.

OpenAI reviews quarterly. Response time has historically been 4–8 weeks
after the quarterly cutoff.

## What to attach if the form supports uploads

- `docs/AAIF-PROPOSAL-v3.md` as PDF (background)
- `formal/PROOF_STATUS.md` (proof of formal verification claim)

If the form doesn't accept attachments, link them as raw GitHub URLs in
section 8. OpenAI reviewers do follow public-artifact links.

## What to expect post-submission

- Auto-acknowledgement email within ~24h
- If shortlisted: short interview / video call within 4–8 weeks
- Decision communicated by email; awards delivered as credits + cash
  (split varies by grant)

## What to do BEFORE submitting

OpenAI's program prioritizes projects that are already shipping.
Make sure these are public on the day you submit:
- [x] `@emilia-protocol/verify` on npm (already published)
- [x] Live demo / playground at emiliaprotocol.ai
- [x] Public GitHub with active commits in the last 30 days
- [ ] One named external reference / advisor — not strictly required but
  strengthens the application. Worth 30 min asking someone in your network
  (NIST AISI contact, AAIF reviewer, AI safety researcher) if they'll
  permit being named as an advisor.
