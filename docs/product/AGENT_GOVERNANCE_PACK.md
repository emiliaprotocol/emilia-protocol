# Agent Governance Pack

The Agent Governance Pack applies EP to agent-assisted and autonomous workflows where broad access must be replaced with action-level control.

## First workflows

- destructive tool use
- delegated principal actions
- irreversible customer-impacting actions
- sensitive operational changes

## What it contains

- action risk classes
- signoff thresholds
- tool-use control patterns
- example escalation paths from Emilia Eye into Handshake and Signoff
